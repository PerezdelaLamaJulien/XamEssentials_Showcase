Projet XamEssentials Showcase.
	Créé par Julien Perez pour Davidson

	But du projet : Ce projet a pour but de montrer les différentes fonctionnalités du plugin Xamarin Essentials.

	Liste des Fonctionnaltés : 
		- App Actions : Permet la création d'App Actions, des liens qui permettent d'aller directement à une fonctionnalité précise.  Ces liens sont accessible dans l'interface ANdroid et iOS
		- Contacts : Permet de recevoir les informations d'un contact enregistré dans l'annuaire du téléphone.
		- DetectShake : Permet de détecter si l'utilisateur secoue son téléphone.
		- Email : Permet d'écrire un mail, avec ses destinataires, son objet et son corps, avant de redirigé vers l'application par défaut enregistré dans le téléphone pour l'envoyer.
		- FlashLight : Permet d'allumer ou d'éteindre la lampe torche du téléphone.
		- Géolocation & Geocoding : Permet de recevoir les coordonnées géographiques de la position actuelle ou d'une adresse donnée.
		- Informations : Permet de recevoir des informations diverses sur le téléphone, son système d'explotation et l'application.
		- Launcher : Permet de savoir si le téléphone peut redirigé un lien donné vers une application installée et faire le redirection.
		- Maps : Permet d'ouvrir l'application par défaut de Cartes (Google Maps par exemple), avec une adresse donnée et avec des marqueurs.
		- MediaPicker : Permet de prendre ou de choisir dans la gallerie des photos et des vidéos enregistrées dans le téléphone.
		- Metrics :
		- Permet d'ouvrir un explorateur internet avec une adresse donnée.
		- Permissions : Permet de vérifier et de demander à l'utilisateur une permission donnée (exemple accès à la caméra, les fichiers, les contacts ect...)
		- PhoneDialer : Permet d'ouvrir l'application Téléphone avec un numéro donné
		- Préférences : Permet d'enregistrer des données simples (chaine de caractères, boolean, nombre entiers), afin des les retrouver lors d'une future utilisation de l'application.
		- ScreenShot : Permet de prendre un screenshot de l'application
		- SecureStorage : Permet d'enregistrer des données de manière sécurisée (avec hashage et cryptage des données),afin de les retrouver lors d'une future utilisation
		- Share : Permet de partager un texte ou un lien par différents moyens (SMS, Mails, Reseaux Sociaux)
		- SMS : Permet d'écrire un SMS avec ses destinataires et son texte, avant de redirigé vers l'application par défaut enregistrée dans le téléphone pour l'envoi'
		- TextToSpeech : Permet d'écouter un audio d'une chaine de caractère donnée.
		- Vibrate : Permet de faire vibrer le téléphone
